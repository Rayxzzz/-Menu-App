# Cashier App

**Cashier App** adalah aplikasi untuk memudahkan cashier dalam penghitungan biaya, pemilihan menu dan penambahan menu


## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pilarhh/blanja-app.git
   ```

2. Run with Live Server
